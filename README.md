# Portfolio Basic

Primeiro projeto desenvolvido durante meus estudos iniciais em Desenvolvimento Web. O objetivo foi aplicar fundamentos de HTML e CSS na construção de uma página de portfólio simples, com foco em estruturação semântica, organização visual e boas práticas básicas de front-end.

Este repositório marca o início da minha transição para a área de tecnologia e representa minha primeira entrega prática publicada no GitHub.

## Tecnologias

* HTML5
* CSS3

## Funcionalidades

* Estrutura de página pessoal
* Página de contato
* Estilização com CSS puro
* Organização básica de layout

## Estrutura

* index.html — página principal
* contato.html — formulário de contato
* style.css — estilos do projeto
* image.png — imagem utilizada no layout

## Objetivo do Projeto

Praticar fundamentos essenciais do desenvolvimento web:

* Estruturação semântica
* Separação entre conteúdo e estilo
* Organização de arquivos
* Publicação com GitHub Pages

## Como executar

Clone o repositório e abra o arquivo index.html no navegador:

git clone https://github.com/sibelecaetano/portfolio-basic.git

## Próximos passos

Este projeto faz parte da evolução do meu portfólio front-end. Versões futuras incluem JavaScript, responsividade avançada e melhorias de acessibilidade.
